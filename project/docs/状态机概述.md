## 4. 关键转移条件

### 4.1 NORMAL -> CROSS_1

要求：

1. 十字检测开关开启
2. 左右下角点都找到且索引有效
3. 左右角点后沿边框连续行数都 >= route_cross_entry_corner_post_frame_wall_rows_min

### 4.2 NORMAL -> STRAIGHT（连续帧）

要求 straight_state_ready 连续满足指定帧数：

1. straight_required_last_index >= 0
2. selected_centerline_count > straight_required_last_index
3. straight_abs_error_sum < route_straight_abs_error_sum_max
4. 连续帧数 >= route_straight_enter_consecutive_frames

### 4.3 CROSS_1 -> CROSS_2

要求：

1. left_start_frame_wall_rows >= route_cross_stage2_enter_start_frame_wall_rows_min
2. right_start_frame_wall_rows >= route_cross_stage2_enter_start_frame_wall_rows_min

### 4.4 CROSS_2 -> NORMAL

当前已按最新代码调整为仅看起始行 gap：

1. start_boundary_gap_x > 0
2. start_boundary_gap_x < route_cross_exit_start_gap_x_max

不再要求左右起始行都不贴边。

### 4.5 圆环主转移（左右镜像）

左圆环：

1. LEFT_1 -> LEFT_2: left_start_frame_wall_rows >= route_circle_stage_frame_wall_rows_enter
2. LEFT_2 -> LEFT_3: left_start_frame_wall_rows <= 0
3. LEFT_3 -> LEFT_4: right_start_frame_wall_rows > route_circle_stage3_frame_wall_rows_trigger
4. LEFT_4 -> LEFT_5: right_corner_found
5. LEFT_5 -> LEFT_6: right_start_frame_wall_rows >= route_circle_stage_frame_wall_rows_enter
6. LEFT_6 -> NORMAL: left_straight && right_straight

右圆环按左右镜像。

## 5. 输入与输出

输入结构见 [project/code/driver/vision/vision_route_state_machine.h](project/code/driver/vision/vision_route_state_machine.h#L40)：

4. 圆环入口 raw gap 布尔量

输出快照见 [project/code/driver/vision/vision_route_state_machine.h](project/code/driver/vision/vision_route_state_machine.h#L69)：

1. main_state
2. sub_state
3. preferred_source
4. encoder_since_state_enter
5. cross/left/right loss/gain 计数
