# 圆环状态机（按当前代码）

## 1. 实现位置

1. 状态机： [project/code/driver/vision/vision_route_state_machine.cpp](project/code/driver/vision/vision_route_state_machine.cpp)
2. 子状态补线： [project/code/driver/vision/vision_image_processor.cpp](project/code/driver/vision/vision_image_processor.cpp)
3. 参数配置： [project/code/driver/vision/vision_config.c](project/code/driver/vision/vision_config.c)

当前圆环状态机是主链生效逻辑，不是预留代码。

## 2. 入口条件

### 2.1 NORMAL -> CIRCLE_LEFT

left_circle_entry_ready 要求：

1. right_straight
2. left_corner_found
3. left_corner_y > 60
4. left_corner_index >= 0
5. right_boundary_count > route_circle_entry_min_boundary_count
6. left_circle_entry_raw_gap_ok
7. left_corner_index < right_boundary_count - route_circle_entry_corner_tail_margin

### 2.2 NORMAL -> CIRCLE_RIGHT

right_circle_entry_ready 对称条件：

1. left_straight
2. right_corner_found
3. right_corner_y > 60
4. right_corner_index >= 0
5. left_boundary_count > route_circle_entry_min_boundary_count
6. right_circle_entry_raw_gap_ok
7. right_corner_index < left_boundary_count - route_circle_entry_corner_tail_margin

## 3. 子状态推进

### 3.1 左圆环

1. CIRCLE_LEFT_1 -> CIRCLE_LEFT_2
条件：left_start_frame_wall_rows >= route_circle_stage_frame_wall_rows_enter

2. CIRCLE_LEFT_2 -> CIRCLE_LEFT_3
条件：left_start_frame_wall_rows <= 0

3. CIRCLE_LEFT_3 -> CIRCLE_LEFT_4
条件：right_start_frame_wall_rows > route_circle_stage3_frame_wall_rows_trigger

4. CIRCLE_LEFT_4 -> CIRCLE_LEFT_5
条件：right_corner_found

5. CIRCLE_LEFT_5 -> CIRCLE_LEFT_6
条件：right_start_frame_wall_rows >= route_circle_stage_frame_wall_rows_enter

6. CIRCLE_LEFT_6 -> NORMAL
条件：left_straight && right_straight

### 3.2 右圆环

右圆环按左右镜像：

1. RIGHT_1 -> RIGHT_2: right_start_frame_wall_rows >= route_circle_stage_frame_wall_rows_enter
2. RIGHT_2 -> RIGHT_3: right_start_frame_wall_rows <= 0
3. RIGHT_3 -> RIGHT_4: left_start_frame_wall_rows > route_circle_stage3_frame_wall_rows_trigger
4. RIGHT_4 -> RIGHT_5: left_corner_found
5. RIGHT_5 -> RIGHT_6: left_start_frame_wall_rows >= route_circle_stage_frame_wall_rows_enter
6. RIGHT_6 -> NORMAL: left_straight && right_straight

## 4. preferred_source 与补线关系

状态机内偏好源逻辑：

1. 左圆环主状态默认 RIGHT，左圆环 3/4/5 阶段临时切到 LEFT
2. 右圆环主状态默认 LEFT，右圆环 3/4/5 阶段临时切到 RIGHT

图像侧补线逻辑：

1. 左圆环 state3/state5 替换右边界
2. 右圆环 state3/state5 替换左边界

因此“补哪边”和“偏好哪边”是刻意反向搭配。

7. circle_guide_min_frame_wall_segment_len = 8
8. circle_guide_target_offset_stage3 = 5
9. circle_guide_anchor_offset_stage5 = 0

## 6. 与控制层联动

line_follow 的参数包选择见 [project/code/app/line_follow_thread/line_follow_thread.cpp](project/code/app/line_follow_thread/line_follow_thread.cpp#L229)：

1. CIRCLE_*_1 -> CIRCLE_ENTER
2. CIRCLE_*_2/3/4 -> CIRCLE_INSIDE
3. CIRCLE_*_5/6 -> CIRCLE_EXIT

所以圆环子状态不仅影响补线，也直接影响控制参数。
