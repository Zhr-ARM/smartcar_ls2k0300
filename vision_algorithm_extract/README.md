# vision_algorithm_extract

该目录从原工程中提取了视觉处理算法相关代码，聚焦图像处理与赛道元素识别逻辑。

## 目录结构
- `src/`: 提取的算法源文件
- `include/`: 对应算法头文件
- `include/deps/`: 为独立阅读/迁移而重建的最小依赖头文件（接口声明与常量）

## 已提取算法文件
- `imgproc.c/.h`: 基础图像处理、巡线、角点/曲率相关点集处理
- `camera_param.c/.h`: 相机标定参数、映射表与逆映射
- `cross.c/.h`: 十字元素检测/通行状态机
- `circle.c/.h`: 环岛元素检测/通行状态机
- `yroad.c/.h`: 三岔路检测/通行状态机
- `garage.c/.h`: 车库(斑马线)识别与进出库状态机
- `apriltag.c/.h`: 黑斑 AprilTag 近似识别逻辑

## 说明
- `include/deps` 不是原工程完整驱动/RTOS 头文件，而是为算法提取场景重建的最小接口。
- 原有硬件相关实现（驱动、系统初始化、RTOS实体定义）未迁移。
